# clock

## HTML5 Canvas + electron

名前と写真の通りただの時計です。Mac でしか正常に表示されないと思われます。

### How to use

```bash
git clone https://github.com/unios103/canvas_clock.git
yarn build
```

→ `./dist/mac/clock.app` または `./dist/clock-1.0.0.dmg`、`clock-1.0.0-mac.zip`
